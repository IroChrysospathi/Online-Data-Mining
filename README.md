# Online Data Mining

## Purpose
This repository contains a group project for scraping online webshop data
(products and optional support information) using Python and Scrapy.

## Scope
- Scrape multiple online shops
- Store scraped data as raw files and/or in a database
- Provide a reproducible scraping pipeline

## Team Contributions
Each team member is responsible for specific components of the project
(e.g., individual spiders, data model, pipelines).

## How to Run
Instructions for installing dependencies and running spiders are provided.

## Notes
This project is intended for educational purposes only.
